Abstract

Traditional campus surveillance relies entirely on human operators monitoring multiple CCTV feeds simultaneously — a task fundamentally limited by fatigue, divided attention, and the sheer volume of footage. Incidents frequently go undetected until significant harm has occurred.

This project presents HAWK-EYE V3, an autonomous AI-powered security monitoring system that replaces passive CCTV with an intelligent, always-on detection pipeline. At the edge, OpenCV performs lightweight frame differencing to detect motion — acting as a zero-cost gatekeeper that fires the AI only when activity is detected.

When triggered, a Google Gemini 2.5 Pro multimodal agent receives a temporal sequence of three frames spaced two seconds apart and reasons through the scene using a ReAct-style loop — querying past incident history, checking time context, reviewing officer feedback, then deciding to log an incident, link to an existing case, or mark the scene clear.

The system detects 20 threat categories specific to Indian college campuses, persists all incidents to MongoDB Atlas with full image evidence and agent reasoning chains, dispatches email alerts to authorities, and provides a real-time web dashboard for security operations — all without requiring continuous human attention.
